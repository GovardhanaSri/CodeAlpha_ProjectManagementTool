# CodeAlpha Project Management Tool

## 📌 Overview

The CodeAlpha Project Management Tool is a modern web application designed to help teams organize projects, manage tasks, and collaborate efficiently. It provides an intuitive dashboard for tracking project progress and managing work assignments.

## 🚀 Features

* User Authentication
* Dashboard
* Create, Edit and Delete Projects
* Task Management
* Task Status Tracking
* Project Progress
* Responsive UI
* Modern Design

## 🛠️ Tech Stack

### Frontend

* React.js
* TypeScript
* HTML5
* CSS3
* Tailwind CSS
* Vite

### Backend

* Supabase

## 📂 Project Structure

* src/
* components/
* pages/
* services/
* hooks/
* supabase/
* assets/

```bash
npm install
npm run dev
```
